### 2) Gold Set 规则（已冻结）
> 你问的“Gold Set 是什么”——它就是**评测时的标准答案集**，用于计算召回率。

#### 定义
Gold Set = 某目标公司“确认归属”的资产集合（MVP 先以域名/子域为主）。

#### 纳入标准（满足任一强证据）
- 在官方 bug bounty scope 中明确出现；
- 在目标公司官方文档/官网/开发者站点明确声明归属；
- 有双证据闭环（例如：证书主体 + 官方页面品牌/备案/公司声明）。

#### 排除标准
- 仅单弱信号（仅一次 DNS 关联、仅论坛/第三方帖子提及）；
- 无法复核来源或证据链断裂；
- 明显第三方平台托管且无法证明归属。

#### 版本策略
- 使用 `dataset_version` 管理；
- 每次评测冻结 test_set，不在同一轮中途改答案。
